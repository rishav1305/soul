This Pattern

  • All 19+ products visible at once — no hidden menus
  • Grouped by category with separators
  • Scrollable when products exceed viewport
  • Collapse to 56px icon rail for max content space
  • ⌘K search built into expanded sidebar
  • Hover tooltips on collapsed icons
  • Active item highlighted with gradient accent

Considerations

  • Mobile: sidebar becomes bottom sheet or hamburger overlay
  • Chat sessions sidebar needs to nest inside chat page (not global)
  • Persist collapsed/expanded state in localStorage
  • Smooth CSS transition between states (200ms ease)
