Anatomy — What Lives Where
① Quick Actions (top)
New Chat — prominent golden button
Search — icon button next to it
Always visible, never scrolled away
② Product Switcher (compact)
Shows top 4 products + expandable "more"
Tapping a product: sets active product AND closes sidebar
Active product shown with golden dot + highlight
This replaces the product selector in ChatInput on mobile
③ Session List (scrollable)
Replaces top bar buttons: All / Running / Unread as filter tabs

Active session: left gold border + brighter text

Interaction Flow

User lands on chat → sees: [☰] Session Title [⊕]

Tap ☰ OR swipe right from edge:
  → Sidebar slides in (280px, 85vw max)
  → Backdrop dims main content (tap to close)

In sidebar:
  → Tap "New Chat" → creates session, closes sidebar
  → Tap product → sets product, closes sidebar
  → Tap session → switches session, closes sidebar
  → Tap filter (Running/Unread) → filters session list
  → Tap search → opens search overlay

Swipe left on sidebar OR tap backdrop → closes sidebar
