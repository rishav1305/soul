Layout Breakdown

What each section does

Pillar Health Strip

Top row shows all 6 pillars with pass/total count. Color-coded: green = all pass, amber = warnings, red = failures. Always visible regardless of tab.

Overview Tab

System status (uptime, sessions, events), cost summary, and active alerts. The "at a glance" view combining status + cost + alerts CLI reports.

Pillar Tabs (6)

Each pillar tab lists its constraints with live metric values. Green/amber/red border per constraint. Maps directly to PILLARS.md definitions.

Tail Tab

Scrollable event log — latest events at top, filterable by event type prefix. Manual refresh to load new events.
