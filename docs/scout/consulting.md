## Scope

Expert consulting: you give ADVICE. Three tiers from 1-hour calls to project engagements. Each tier feeds the next, and consulting feeds the contracts pipeline.

## Three Tiers

| Tier | What | Rate | Time |
|---|---|---|---|
| **Expert Calls** | 1-hr calls via GLG, Guidepoint, etc. | ₹10-30K/hr | As scheduled |
| **Advisory** | Monthly advisor retainer | ₹50K-2L/month | 2-5 hrs/month per client |
| **Project Consulting** | AI audit, roadmap, vendor eval — deliverable, not code | ₹2-10L/project | 10-40 hrs/project |

## Expert Domains

**Horizontal:** AI/LLM production, AI strategy & architecture, AI market landscape
**Vertical:** Legal AI, Healthcare AI, Sales AI, E-commerce AI

## Onboarding (Week 1-4)

- Week 1-2: Mass-apply to 12 expert networks (AI tailors each application)
- Week 3-4: Optimize profiles with specific topics per vertical
- Month 2+: First calls start (expect 6-8 acceptances out of 12)

## AI Pipeline (2 Gates)

```
IDENTIFY (network calls + TheirStack "AI confusion" signals + cross-pipeline upsells)
  → PREPARE (AI generates call prep brief / advisory proposal / project proposal)
  → ★ Gate E1: Accept/decline calls, review proposals
  → DELIVER (you handle calls/sessions)
  → ★ Gate E2: Post-engagement — follow-up, upsell evaluation, testimonial, referral
```

## Progression Funnel

```
Expert call → advisory retainer → project consulting → team contract
Each tier feeds the next. Consulting is the top-of-funnel for contracts.
```
