## Operating Model

AI works continuously → queues outputs → you verify gates asynchronously when available. No rigid time blocks. Seven days a week. Adaptive trust over time.

## Daily Rhythm

| Block | Time (IST) | Focus | Your Time |
|---|---|---|---|
| **Morning** | 8-10 AM | Review AI outputs, publish content, send outreach | 45-60 min |
| **Midday** | 12-2 PM | Engage comments, submit applications, send proposals | 20-30 min |
| **Afternoon** | 2-8 PM | PAID WORK (freelance, contracts, consulting) | 4-6 hrs |
| **Evening** | 9-10 PM | Log interactions, plan, research | 15-20 min |

## Weekly Day-by-Day

| Day | Theme | Key AI Outputs | Key You Actions |
|---|---|---|---|
| **Mon** | Discovery | TheirStack sweep, gig scoring, content series, contract targets | Gate 1, C1, P1, F1 reviews |
| **Tue** | Outreach | Networking drafts, X replies | Publish Part 1, Gate N1, F1 |
| **Wed** | Follow-up | Connection follow-ups, cadence checks | X thread, Gate 2, F1 |
| **Thu** | Application | Outreach drafts, cold emails | Publish Part 2, Gate N1, applications |
| **Fri** | Review | Networking brief, content pulse, stale leads | Friday Review Session (30 min) |
| **Sat** | Content+Build | Comment reply drafts | Publish Part 3, GitHub work |
| **Sun** | Planning | Topic suggestions, backlog refresh | Pick topic, deep work |

## Adaptive Trust

| Month | Review Level | Admin Time |
|---|---|---|
| Month 1 | Everything | ~12 hrs/week |
| Month 2-3 | Selective (all T1 + high-value, spot-check rest) | ~9 hrs/week |
| Month 4+ | Trust with weekly quality audits | ~6 hrs/week |

## Weekly Time Budget

| Type | Hrs/week |
|---|---|
| Admin (gates, reviews, planning) | 9-12 (reduces to 6) |
| Paid work (freelance + contracts + consulting) | 12-27 |
| Investment (deep work, GitHub, research) | 4-8 |
| **TOTAL** | **25-47** |

AI: ~200-400 automated actions/week at ₹0 cost (Claude OAuth).
