## 3-Phase Discovery

### Phase 1: Platform Bootstrapping (Month 1-2)
- Upwork → Contra → Freelancer (priority order)
- 3-5 AI-drafted proposals/day, score > 80 only
- Goal: 5 reviews in 6-8 weeks → Rising Talent / Top Rated

### Phase 2: Networking Conversion (Month 2-4)
- Section 4 founder/CTO contacts → freelance angle
- Vetted networks accept you (Toptal, Flexiple)
- Reduce platform proposals to 2-3/day

### Phase 3: Inbound Takes Over (Month 4+)
- Content (Section 6) → "I built X" → client DMs
- LinkedIn Services, referrals, vetted network gigs
- Platform proposals: 1-2/week (cherry-pick only)

## AI Pipeline (2 Gates)

```
DISCOVER → SCORE → DRAFT (automated)
  ★ Gate F1: Proposal Review (15 min daily) — [Send] [Edit] [Skip]
EXECUTE → DELIVER → COMPLETE
  ★ Gate F2: Post-Project (per completion) — review, upsell, referral
```

## Pipeline Stages

```
found → proposal-ready → proposal-sent → shortlisted → awarded → delivering → completed
Terminal: completed, lost, withdrawn
```
