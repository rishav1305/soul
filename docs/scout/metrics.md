## Metrics Framework

Every section tracks **leading** (did I do the work?) and **lagging** (did it produce results?) metrics. Leading metrics should be hit weekly. Lagging metrics are watched but not stressed in Month 1-2.

## Review Cadence

| Review | When | Time | What |
|---|---|---|---|
| **Daily** | Morning | 2 min | Priority Queue status scan |
| **Weekly** | Friday review session | 30 min | Gate N2 (networking brief) + P-metrics (content pulse) + Gate 3 (stale leads) |
| **Monthly** | End of month | 1 hr | Content retrospective + pillar rebalancing + revenue review + next month calendar |
| **Quarterly** | Every 3 months | 2 hrs | Gate PR2 (profile audit) + vertical rotation assessment + rate adjustment + strategy pivot decisions |

### Combined Revenue Trajectory

| Period | Freelance | Contracts | Consulting | Total |

| Month 6+ | ₹3-5L | ₹8-15L | ₹1.5-5L | ₹7.5L-18L+ |

### Conversion Funnels

**Jobs:**
```
Leads discovered → Qualified (40%) → Prepared (30%) → Applied (20%) →
Response (10%) → Interview (5%) → Offer (2%)

With warm/referral: Response (40%) → Interview (25%) → Offer (10%)
```

**Freelance:**
```
Gigs discovered → Scored >80 (30%) → Proposal sent (20%) →
Shortlisted (8%) → Awarded (4%) → Completed (3.5%)
```

**Networking:**
```
Month 1: 50 contacts → 40 NEW, 8 ACTIVE, 2 WARM
Month 2: 100 contacts → 50 NEW, 30 ACTIVE, 15 WARM, 5 READY
Month 3: 150 contacts → 50 NEW, 40 ACTIVE, 35 WARM, 20 READY
```

## Dashboard Design

### Priority Queue Tab (default view)

All pending actions ranked by urgency:
- Urgent: expert calls, client messages, inbound hot leads
- High: Tier 1 outreach, content publish, time-sensitive proposals
- Medium: applications, proposals, networking follow-ups
- Low: profile updates, stale leads, backlog management

### Per-Section Tabs

| Tab | Key Widgets |
|---|---|
| **Jobs** | Leads by tier, pipeline funnel, application tracker, follow-up reminders |
| **Networking** | Warmth dashboard, channel breakdown, coffee chat tracker, conversion opportunities |
| **Freelance** | Proposal queue, active gigs, revenue tracker, reviews count |
| **Contracts** | Target pipeline, active negotiations, upsell alerts, team utilization |
| **Consulting** | Upcoming calls + prep, advisory schedule, network acceptance status |
| **Content** | Publish calendar, metrics dashboard, backlog queue, engagement tracker |
| **Profile** | Pending updates, quarterly audit, GitHub activity graph, testimonial tracker |
