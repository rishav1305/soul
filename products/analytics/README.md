# Analytics — Soul Product

Data analytics pipeline automation

## Status

🚧 **Scaffolded** — Core structure in place, implementation pending.

## Architecture

This product follows the Soul product pattern:
- gRPC service implementing `ProductServiceServer`
- Communicates with Soul core via Unix socket
- Tools exposed via `GetManifest` and `ExecuteTool`

## Tools

| Tool | Description |
|------|-------------|
| `analyze` | Analyze target for analytics insights |
| `report` | Generate analytics report |

## Development

```bash
# Build
go build -o analytics-go .

# Run (standalone)
./analytics-go --socket /tmp/analytics.sock
```
